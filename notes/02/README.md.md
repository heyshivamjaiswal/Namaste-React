### What is `NPM`?

`NPM` (Node Package Manager) is a tool used to manage and install packages for Node.js projects.

It comes with Node.js and provides a CLI to interact with the **NPM Registry**, which hosts public and private packages.

- **Initialize NPM project:**
    
    ```bash
    npm init
    
    ```
    
    Use `npm init -y` to skip manual setup and auto-generate `package.json`.
    
- **Alternative:** `yarn`
